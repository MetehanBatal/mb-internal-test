In order to use all of the features and functionality of Element Home's Website, you need to accept cookies. Web Beacons. A web beacon, sometimes called a tracking pixels, is a programming code that can be used to display an image on a web page, but can also be used to transfer your unique user identification to a database and associate you with previously acquired information about an individual in a database. This allows Element Home and our third-party service providers to track certain websites you visit. Web beacons are used to track online behavioral habits for marketing purposes to determine products or services you may be interested in. In addition to using web beacons on web pages, we also use web beacons in email messages sent to individuals listed in Element Home's database. Other Online Tools.
